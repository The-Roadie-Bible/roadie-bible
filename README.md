# The Roadie Bible

A searchable travel guide website for hotels, restaurants, bars, places to watch sport, and things to do around the world.

## How to run locally

1. Install Node.js from https://nodejs.org
2. Open this folder in Terminal / Command Prompt
3. Run:

```bash
npm install
npm run dev
```

## How to deploy on Vercel

1. Upload this folder to a GitHub repository.
2. Go to https://vercel.com
3. Click Add New > Project
4. Import the GitHub repository
5. Click Deploy

## How to edit recommendations

Open:

```text
src/main.jsx
```

Edit the `places` list near the top of the file.
